05 / Полезно знать

Ответы
на вопросы.

Что нужно для запуска StreamNumDeck?+

Windows 10 версии 1903 или новее либо Windows 11. Цифровой блок и шесть навигационных клавиш можно перехватывать независимо.

Работает ли приложение с современной OBS Studio?+

Да. StreamNumDeck использует интерфейс OBS WebSocket 5.x, встроенный в актуальные версии OBS Studio.

Обязательно ли устанавливать приложение?+

Нет. Можно выбрать обычный установщик или скачать portable ZIP, распаковать его и запустить StreamNumDeck.exe.

Почему SmartScreen может показать предупреждение?+

Установщик пока не имеет цифровой подписи. Скачивайте его только из официального репозитория GitHub и при необходимости сверяйте SHA-256.
