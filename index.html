Good to know

Questions,
answered.

What do I need to run StreamNumDeck?+

Windows 10 version 1903 or newer, or Windows 11. A numeric keypad is recommended; the six-key navigation block can be captured independently.

Does it work with current OBS Studio?+

Yes. StreamNumDeck uses the OBS WebSocket 5.x interface built into current OBS Studio releases.

Do I have to install it?+

No. Choose the regular per-user Setup or download the portable ZIP, extract it, and run StreamNumDeck.exe.

How much does it cost?+

It is free for personal and other permitted non-commercial use. Commercial use requires separate permission.

Why can SmartScreen show a warning?+

The installer is not digitally signed yet. Download it only from the official GitHub repository and verify the published SHA-256 hash if needed.
